--------------------
Extra: microcache
--------------------
Version: 1.0.0-beta
Released: April 24, 2013
Author: Jason Coward <jason@modx.com>

The microcache extra is made up of a simple Plugin to set HTTP caching headers, allowing fine-grained control of which Resources are to be sent with which caching headers. This can be configured through the Plugin properties.

Official Documentation at https://github.com/opengeek/microcache/wiki
